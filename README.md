## Ejecucion

En la consola de comandos del proyecto se deben realizar los siguientes comandos para ejecutar el progama

#### Setup proyecto

```bash
$ npm install
```

#### Compilar y ejecutar proyecto

```bash
# desarrollo
$ npm run start

# Modo vista
$ npm run start:dev

# Modo produccion
$ npm run start:prod
```
